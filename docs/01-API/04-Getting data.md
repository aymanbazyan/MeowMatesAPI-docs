# Request data

---

:::note
You must provide **Authorization: "Bearer "TOKEN"** in **Headers** for each request
:::

> ## Get all users

using `GET` at **/api/v1/users**

```bash
<URL>/api/v1/users
```

> ## Get 1 user

using `GET` at **/api/v1/users/;id**

```bash
<URL>/api/v1/users/<123456789>
```
